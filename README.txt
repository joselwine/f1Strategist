Project Overview
This project develops a data-driven system for analysing Formula One race strategy using machine learning and simulation.

How to Run
1. Install dependecies: pip install -r requirements.txt
2. Run the application: python app/ui_streamlit.py

Structure
- src/: core modelling and simulation logic
- app/: user interface and services
- figures/: outputs used in dissertation

Notes
- Data retrieved using FastF1
